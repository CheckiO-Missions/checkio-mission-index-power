üí° The idea is to compare the given index n with the length of the given array ar. If n is inside the ar - return element with the index powered to the same n, otherwise - return -1.

Ok, what should I start from?

Read about conditional (ternary) operator and use it in your code to check, if the given index n is inside the array ar, which means that n should be less than the length of the ar. 

let result = n < ar.length ? ... : ...;

Ok, what to do next?

In case, the condition is true (index is inside the array), your function should return the element of array with this index, powered to the index. 

let result = n < ar.length ? ar[n]**n : ...;

How to complete the conditional operator?

According to the description, if the index is outside the array, your function should return just -1. 

let result = n < ar.length ? ar[n]**n : -1;

So, how my final solution should look like?

You just need to wrap your code into function expression and return the result.

Here is the whole solution:

import assert from "assert";

function indexPower(ar: number[], n: number): number {

    let result = n < ar.length ? ar[n]**n : -1;
    return result
}

This solution is generally the same, but uses anonymous function and Math.pow() function for powering. 

var indexPower = (ar: number[], n: number): number => n < ar.length ? Math.pow(ar[n], n) : -1;

Since Math.pow(ar[n], n) (or ar[n]**n) returns NaN in case index is outside the array, you may use logic operator "OR" || instead of ternary operator. 

var indexPower = (ar: number[], n: number): number => ar[n] ** n || -1;
// or
var indexPower = (ar: number[], n: number): number => Math.pow(ar[n], n) || -1;

Here is a more traditional solution, using if statement. Note, that you may not use else, if you use return inside above statement. You also may not use curly brackets, if there is only one expression inside statement and semicolon after the last expression before closing. 

function indexPower(ar: number[], n: number): number {
    
    if (n < ar.length)
        return ar[n]**n;
    return -1
}
