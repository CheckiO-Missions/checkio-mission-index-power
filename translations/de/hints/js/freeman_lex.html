💡 Die Idee ist, den angegebenen Index n mit der Länge des angegebenen Arrays ar zu vergleichen. Wenn n innerhalb von ar liegt, geben Sie das Element mit dem Index zurück, der auf n liegt, andernfalls geben Sie -1 zurück.

Ok, womit soll ich anfangen?

Lesen Sie über conditional (ternary) operator und verwenden Sie es in Ihrem Code, um zu prüfen, ob der angegebene Index n innerhalb des Arrays ar liegt, was bedeutet, dass n kleiner sein sollte als die Länge von ar. 

let result = n < ar.length? ...: ...;

Ok, was ist als nächstes zu tun?

Falls die Bedingung true lautet (der Index befindet sich innerhalb des Arrays), sollte Ihre Funktion das Element des Arrays mit diesem Index zurückgeben, das mit dem Index verbunden ist. 

let result = n < ar.length? ar[n]**n: ...;

Wie vervollständige ich den bedingten Operator?

Laut der Beschreibung sollte Ihre Funktion nur -1 zurückgeben, wenn der Index außerhalb des Arrays liegt. 

let result = n < ar.length? ar[n]**n: -1;

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur Ihren Code in einen Funktionsausdruck einpacken und das Ergebnis zurückgeben.

Hier ist die gesamte Lösung:

import assert from "assert";

function indexPower(ar: number[], n: number): number {

    let result = n < ar.length? ar[n]**n: -1;
    return result
}

Klar, schnappen Sie es sich!

Diese Lösung ist im Großen und Ganzen die gleiche, verwendet aber eine anonyme Funktion und die Funktion Math.pow() für die Stromversorgung. 

var indexPower = (ar: number[], n: number): number => n < ar.length? Math.pow(ar[n], n): -1;

Da Math.pow(ar[n], n) (or ar[n]**n) NaN zurückgibt, wenn der Index außerhalb des Arrays liegt, können Sie den logischen Operator "OR" || anstelle des ternären Operators verwenden. 

var indexPower = (ar: number[], n: number): number => ar[n] ** n || -1;
// or
var indexPower = (ar: number[], n: number): number => Math.pow(ar[n], n) || -1;

Hier ist eine traditionellere Lösung, die die Anweisung if verwendet. Beachten Sie, dass Sie else nicht verwenden dürfen, wenn Sie return innerhalb der obigen Anweisung verwenden. Sie dürfen auch keine geschweiften Klammern verwenden, wenn es nur einen Ausdruck innerhalb der Anweisung gibt und ein Semikolon nach dem letzten Ausdruck vor dem Schließen. 

function indexPower(ar: number[], n: number): number {
    
    if (n < ar.length)
        return ar[n]**n;
    return -1
}
