Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Po pierwsze, aby uzyskać n-ty element tablicy można użyć array[n]. Proszę poczytać o lists indexes.

Potrzebuję pomocy, aby kontynuować misję.

Musi Pan sprawdzić, czy wymagany element znajduje się wewnątrz tablicy, używając instrukcji warunku i funkcji len(). W przeciwnym razie proszę zwrócić -1. 

if n < len(array):
    return ...
return -1

Jestem w połowie drogi. Proszę o pomoc!

Musi Pan podnieść n-ty pierwiastek do n-tej potęgi. Do obliczenia potęgi proszę użyć operatora **. 

2**3 == 8
...
array[n]**n

Ostatnie pytanie. Jak będzie wyglądało moje ostateczne rozwiązanie?

Musi Pan po prostu zawinąć swój kod w wyrażenie funkcyjne z podpowiedziami typu i zwrócić obliczony wynik.

def index_power(array: list[int], n: int) -> int:   

    if n < len(array):
        return array[n]**n
    return -1
