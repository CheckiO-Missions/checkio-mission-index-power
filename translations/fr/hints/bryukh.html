Une dernière question. À quoi ressemblera ma solution finale?

Il vous suffit d'envelopper votre code dans une expression de fonction avec des indications de type et de renvoyer le résultat calculé.

def index_power(array: list[int], n: int) -> int:   

    if n < len(array):
        return array[n]**n
    return -1
