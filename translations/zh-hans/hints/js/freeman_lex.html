好的，我应该从什么开始？

阅读关于conditional (ternary) operator ，并在你的代码中使用它来检查，如果给定的索引n 是在数组ar 内，这意味着n 应该小于ar 的长度。 

let result = n < ar.length? ...: ...;

好的，接下来该怎么做？

如果条件是true （索引在数组内），你的函数应该返回具有该索引的数组元素，并向该索引供电。 

let result = n < ar.length? ar[n]**n: ...;

如何完成条件运算符？

根据描述，如果索引在数组之外，你的函数应该只返回-1 。 

let result = n < ar.length? ar[n]**n: -1;

那么，我的最终解决方案应该是什么样子的呢？

你只需要把你的代码包装成函数表达式并返回结果。

下面是整个解决方案：

import assert from "assert";

function indexPower(ar: number[], n: number): number {

    let result = n < ar.length? ar[n]**n: -1;
    return result
}

这个解决方案大致相同，但使用匿名函数和Math.pow() 函数来供电。 

var indexPower = (ar: number[], n: number): number => n < ar.length? Math.pow(ar[n], n): -1;

由于Math.pow(ar[n], n) (or ar[n]**n) ，如果索引在数组之外，则返回NaN ，你可以使用逻辑运算符 "OR "|| ，而不是三元运算符。 

var indexPower = (ar: number[], n: number): number => ar[n] ** n || -1;
// or
var indexPower = (ar: number[], n: number): number => Math.pow(ar[n], n) || -1;

下面是一个更传统的解决方案，使用if 语句。注意，如果你在上述语句中使用return ，就不能使用else 。如果语句中只有一个表达式，你也可以不使用大括号，而在最后一个表达式后使用分号来结束。 

function indexPower(ar: number[], n: number): number {
    
    if (n < ar.length)
        return ar[n]**n;
    return -1
}
