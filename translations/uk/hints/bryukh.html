Перш за все, для отримання n-го елементу масиву можна використати array[n]. Почитайте про lists indexes.

Мені потрібна допомога для продовження місії.

Вам потрібно перевірити, чи знаходиться потрібний елемент всередині масиву, використовуючи оператор умови та функцію len(). У протилежному випадку повернути -1. 

if n < len(array):
    return ...
return -1

Я не можу пройти половину шляху. Допоможіть!

Потрібно піднести n-й елемент до n-го степеня. Для обчислення степеня використовується оператор **. 

2**3 == 8
...
array[n]**n

І останнє питання. Як буде виглядати мій остаточний розв'язок?

Вам просто потрібно обернути ваш код у вираз з підказками типів і повернути обчислений результат.

def index_power(array: list[int], n: int) -> int:   

    if n < len(array):
        return array[n]**n
    return -1
